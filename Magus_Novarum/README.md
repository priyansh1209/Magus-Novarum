# How to Run the Game
Choose your path to enter the wizard world:

### Option A: Compile from Source (Recommended)
_Best for developers. Safest method, always up-to-date._
#### 1. **Double-click** the `Compile.bat` file in the root folder to compile.
#### 2. **Open** the generated `build` folder.
#### 1. **Double-click** `Game.exe` to begin your magical journey.

### Option B: Run Pre-Compiled Binary
_Best for quick play. No compiler or setup required._
#### 1. **Open** the `bin` folder.
#### 2. **Double-click** the Pre-Compiled `Game.bin` to begin your magical journey.
